# gd_vue_components

gd's vue-components.

## 安装

```
这里建议先删除依赖
cnpm install
cnpm install gd_vue_components --save

```

## Import

```
import 'gd_vue_components/dist/gd-components.min.css';
import components from 'gd_vue_components'
vue.use(components)
```

## Demo

下载该项目，在文件根目录下依次输入命令行：

```
cnpm install
npm run dev
```

然后在浏览器里输入`http://localhost:8089`，即可看到组件 demo。
