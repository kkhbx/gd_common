    # 分片上传组件 gd-slice-upload

    分片上传,针对大文件采用的上传方式,提供自定义分片上传文件,上传成功之后可自行处理业务逻辑, 点击分片上传弹出上传框,然后可以上传文件,只能单个上传

    # 基本用法

  <div>
    <gd-slice-upload
      ref="sliceUpload"
      :size="size"
      :btnName="btnName"
      :token="token"
      @uploadComplete="uploadComplete"
      :actionUrl="actionUrl"
      :uploadIp="uploadIp"
      :composeUrl="composeUrl"
      :chunkSize="chunkSize"
