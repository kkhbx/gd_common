# 步骤条 组件

引导用户按照流程完成任务的分步导航条

# 用法

```
<gd-steps :data="stepsData" :current="currentStep" />

```

## 属性

| 参数    | 说明       | 类型   | 可选值 | 默认值 |
| ------- | ---------- | ------ | ------ | ------ |
| data    | 传入的数组 | Array  | --     | []     |
| current | 绑定步骤   | number | --     | 0      |
