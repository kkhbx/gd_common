# 组件文档编写

组件测试通过之后以及发布成功之后需要完善组件文档

进入项目根目录 gd_components,运行命令 npm run docs:dev

npm run docs:dev
1

运行成功之后,在浏览器访问 http://localhost:8888

对于你刚刚发布的组件是 ui 组件还是工具函数或是对应的 echart?

需要在对应的文件夹中创建文档,

找到 sidebar.js 文件,在此文件上添加对应的名称

举例说明:添加一个组件文档编写,添加 createdDocs
